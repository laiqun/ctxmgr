===============
 CtxMgr 说明
===============

----------
 软件简介
----------

CtxMgr是一款轻便的上下文工具。随时按下快捷键(缺省为Alt+Z)可调出它，
再次按下相同快捷键或ESC键则可令其隐藏、回到系统托盘中。您可在日常
工作中使用该软件快速地记下瞬间灵感、工作备忘、或者电话号码。

和Windows自带的记事本不同的是，CtxMgr会自动保存您输入的全部内容，
并支持多页(TAB)显示。

此外，CtxMgr 支持数学计算，输入“1+2”，按下 Ctrl+Enter，即可出
现计算结果，方便您进行简单的日常计算。

选中文字，按Ctrl+q，即可将选中的文字发送到 CtxMgr 中。


# 数值计算支持以下函数：
+---------------+------------------------------------------------------+---------------------+--------+--+
| 函数名        | 描述                                                   | 用法                 | 结果   |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Abs           | 返回指定数字的绝对值。                                   | Abs(-1)             | 1d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Acos          | 返回余弦值为指定数字的角度。                              | Acos(1)             | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Asin          | 返回正弦值为指定数字的角度。                              | Asin(0)             | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Atan          | 返回正切值为指定数字的角度。                              | Atan(0)             | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Ceiling       | 返回大于或等于指定数字的最小整数。                         | Ceiling(1.5)        | 2d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Cos           | 返回指定角度的余弦值。                                   | Cos(0)              | 1d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Exp           | 返回 e 的指定次幂。                                     | Exp(0)              | 1d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Floor         | 返回小于或等于指定数字的最大整数。                         | Floor(1.5)          | 1d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| IEEERemainder | 返回一个指定数字除以另一个指定数字的余数。                  | IEEERemainder(3, 2) | -1d    |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Ln            | 返回指定数字的自然对数（以 e 为底）。                      | Ln(1)               | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Log           | 返回指定数字的对数（使用指定底数）。                       | Log(1, 10)          | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Log10         | 返回指定数字的以 10 为底的对数。                          | Log10(1)            | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Max           | 返回两个指定数字中较大的一个。                            | Max(1, 2)           | 2      |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Min           | 返回两个数字中较小的一个。                                | Min(1, 2)           | 1      |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Pow           | 返回指定数字的指定次幂。                                  | Pow(3, 2)           | 9d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Round         | 将值四舍五入到最接近的整数或指定的小数位数。                 | Round(3.222, 2)     | 3.22d  |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Sign          | 返回表示数字符号的值（正数返回1，负数返回-1，零返回0）。      | Sign(-10)           | -1     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Sin           | 返回指定角度的正弦值。                                   | Sin(0)              | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Sqrt          | 返回指定数字的平方根。                                   | Sqrt(4)             | 2d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Tan           | 返回指定角度的正切值。                                   | Tan(0)              | 0d     |  |
+---------------+------------------------------------------------------+---------------------+--------+--+
| Truncate      | 计算数字的整数部分（直接截断小数部分）。                    | Truncate(1.7)       | 1      |  |
+---------------+-----------------------------------------------------------------------------+---------------------+--------+--+
