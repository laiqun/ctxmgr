What Overlay2 does

I run a photo studio in London, and I was looking for software that would enable me to make a semi-transparent floating window on my computer. I wanted to be able to open any image to use as a floating positioning guide over the actual image that I'm shooting.

I use Adobe Lightroom for image capture, and I wanted to be able to float the art-director's layout over my photo in Lightroom, to check that my shot was a good fit. Sometimes I wanted to put another element of a composite over my image to check position and perspective, or I wanted to have an image visible over my Lightroom capture so I could match it.

I couldn't find any software to do this, so I wrote it myself: Overlay2 for Mac and Windows is the result. It works with any other Mac or Windows software. 

Use it over video capture or editing software, or 3D modelling software for a reference, positioning guide, preview or test. Or use it to display an image permanently on your screen, or to compare two images.

Floating the OVERLAY2 Window

Overlay2 creates a window on your screen which can be resized and positioned like any normal window. You can open most types of image file to display, the image can be resized, rotated and moved within the window. The OPACITY of the window can also be adjusted from the Overlay Control panel. Overlay2 - The Manual

What Overlay does

I run a photo studio in London, and I was looking for Mac software that would enable me to make a semi-transparent floating window on my computer. I wanted to be able to open any image to use as a floating positioning guide over the actual image that I'm shooting.

I use Adobe Lightroom for image capture, and I wanted to be able to float the art-director's layout over my photo in Lightroom, to check that my shot was a good fit. Sometimes I wanted to put another element of a composite over my image to check position and perspective, or I wanted to have an image visible over my Lightroom capture so I could match it.

I couldn't find any software to do this, so I wrote it myself: Overlay for Mac is the result. It works with Lightroom, Aperture, ACR, 3D modeling applications or any other OS X or WINDOWS software. 

Overlay2-1 is digital tracing paper for your Mac or PC!

How to use Overlay2.1

Install Overlay

Drag the Overlay application icon to your Applications folder on your Mac. The Windows Setup.exe file will install Overlay2-1 in your Programs folder.

If you want Overlay to appear in your Dock, drag it from your applications folder onto your Dock.

Open an image

Drag an image file onto Overlay2-1, or onto Overlay's Dock icon, or choose File > Open from the menus, or click the "Open..." button in the Overlay Control Panel. 

On a Mac, Overlay can open almost all types of image file, including RAW files from all the cameras supported by Mac OS X, and it can open PDFs and animated GIFs. 

On Windows Overlay2-1 can open JPEG, TIFF, PNG, GIF and Animated GIF files.

Rotate, scale and position it.

The Overlay window can be resized and positioned like any normal window, and the controls in The Overlay Control panel (which can also be repositioned) enables you to scale (resize), rotate, and position your image within the window. You can also control the opacity of the main window, and FLOAT it as required. The Overlay windows continue to show on top of ALL other windows on your Mac unless you hide the main window, reduce its opacity to zero, or quit. Use the menu options in the Image menu to view your image at 100%, or to fit it to the window.

On-Image controls

You can also reposition the overlay image by holding down the SPACEBAR while you mouse over the overlay image window (like in Adobe Photoshop). To control opacity in the main window hold SHIFT while mousing. To rotate the image in the main window hold down ALT while mousing, and to scale (resize) the image in the main window, hold COMMAND(Mac)/CNTRL(Windows) while mousing.

FLOAT the OVERLAY window to work straight through it

You can FLOAT the Overlay image by using the Float button. When the OVERLAY window is floating you can work straight through it to the window or application below. (you can use the control panel to change the positiuon and opacity of the floating Overlay window, but you can't use the on-image controls (see above) while the Overlay window is floating).

Overlay behaves slightly differently to other Apple applications because its windows are ALWAYS visible (otherwise they would not be much use). If you have clicked on a background application and you want to get BACK to Overlay, just click on one of its windows. The Overlay FLOATING window is the same size as its normal counterpart but you can’t interact with it in FLOATING mode; to move or resize it click UNFLOAT first.

Replace, print or save the Overlay image.

To replace the overlay image, open a new file. To print or save the image, use the print or save commands in the File menu.

Keyboard shortcuts and menus

Hide or show the main Overlay window using the Window>Hide menu, or the shortcut key CMND-H. You can also hide the Overlay Control Panel using CMND-K, or from the Window menu. For other keyboard shortcuts, see their menu items. Be careful when you have BOTH Overlay and Control windows hidden - you need to have the Overlay application active (check in the menu bar) to be able to make either of the windows visible again, or click the Control Panel item in the Dock to maximize it.

Preferences

Overlay2-1 allows you to set Preferences for the next time you run it. Open the Preferences panel from the Overlay2-1 menu (Mac) or the Options panel from the Edit menu (Windows). You can save the position and size of both the Controls and Overlay windows, save the settings in the Overlay Controls panel, set whether the Overlay Controls panel will be visible, and set Overlay to automatically open the last file displayed.

Registration

If you buy a registration, we’ll send you a unique serial number. Go to Register in the Overlay2-1 menu (Windows: Help menu) and enter your name and serial number to stop the registration reminders from popping up on your screen.
