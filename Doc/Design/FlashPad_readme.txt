===============
 FlashPad 说明
===============

----------
 软件简介
----------

FlashPad是一款轻便的笔记工具。随时按下快捷键(缺省为Alt+C)可调出它，
再次按下相同快捷键或ESC键则可令其隐藏、回到系统托盘中。您可在日常
工作中使用该软件快速地记下瞬间灵感、工作备忘、或者电话号码。

和Windows自带的记事本不同的是，FlashPad会自动保存您输入的全部内容，
并支持多页(TAB)显示。

此外，FlashPad 支持数学计算，输入“1+2”，按下 Ctrl+Enter，即可出
现计算结果，方便您进行简单的日常计算。

 数学计算
----------

FlashPad支持简单的数学计算功能。在 FlashPad 的任意一页中输入表达式，
按 Ctrl+Enter 键，即可出现计算结果。下面是一些例子：

1 + 2 = 3
(1 + 2) * 3 = 9
5.3 + 2.2 = 7.5
5 % 2 = 1
"abc" + "def" = "abcdef"
(1 + 1 = 2) = True
not True = False

power(2, 3) = 8
sqrt(2) = 1.4142135623731
pi() = 3.14159265358979
pi() * power(3.2, 2) = 32.1699087727595
abs(-3) = 3

ascii('中') = 20013
char(20013) = '中'
chars('abc', 2) = 'abcabc'
substr('12345', 2, 3) = '234'
upper('abc') = 'ABC'
lower('ABC') = 'abc'

date() = '2008-11-23'
time() = '15:26:12'
now() = '2008-11-23 15:26:12'
